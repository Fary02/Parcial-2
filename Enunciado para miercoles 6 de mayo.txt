Ejercicio 2

Desarrolle en python un programa

que permita ingresar 2 numeros enteros

el 1 debe ser menor al 2

luego el programa debe generar un programa al azar

entre el rango de los numeros

para hacer esto sigue lo siguiente:

import from random (randint)

numero = randint (n1, n2)

la linea 1 le permite cargar la function randint

la linea 2 usa la function randint() que permite generar un numero aleatorio entre los 2 valores

a modo de ejemplo randint(1, 10). genero un numero aleatorio entre ambos numeros

un generador de numero aleatorio debe ajustar el numero para que el valor final sea adivinado

para el usuario luego de 3 intentos

nota: si el numero ajustado llega a quedar fuera del rango, entonces el numero final debe

dividirse por el limite inferior
